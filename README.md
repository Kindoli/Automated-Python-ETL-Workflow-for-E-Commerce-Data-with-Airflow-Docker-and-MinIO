# Automated-Python-ETL-Workflow-for-E-Commerce-Data-with-Airflow-Docker-and-MinIO
Built an automated Python-based ETL pipeline using Airflow, Docker, and MinIO. The system extracts raw e-commerce data, cleans and aggregates it with Pandas, and stores the processed output in S3-compatible MinIO for analytics. Demonstrates orchestration, containerization, and scalable data processing
